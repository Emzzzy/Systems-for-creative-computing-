What is FLOSS?

Free/Libre open Source Software - free to edit, non-paid, open to everyone 
FLOSS is a an ethical computing software 

The Four essential freedoms 
- Free to run the program 
- the freedom to study how the program works and change it how you wish (access to source code is required)
- The freedom distribute copies to help others 
- The freedom to distribute copies of your modified versions to others
